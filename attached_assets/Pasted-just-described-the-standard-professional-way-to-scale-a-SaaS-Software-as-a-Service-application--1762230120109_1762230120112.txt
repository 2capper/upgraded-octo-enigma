The challenge you've identified is that your Tournament App has two different audiences:

Audience 1 (Private): An Admin from "LaSalle Baseball" who logs into app.dugoutdesk.ca to create and manage their tournament.

Audience 2 (Public): A Coach from "Windsor Hawks" (who doesn't have an account) who just wants to find and view the LaSalle tournament.

You can and should serve both from the same system. The solution is to add a simple "Visibility" setting to your tournaments.

Here is the complete game plan to achieve this.

The "Public Directory" Game Plan
This plan lets your app be both a private admin tool and a public directory.

Step 1: Add "Visibility" to Your Database
In your Drizzle schema, add one new column to your tournaments table. This is the "master switch."

TypeScript

// drizzle/schema.ts
export const tournaments = pgTable('tournaments', {
  // ... all your existing columns like id, name, organizationId ...
  
  // --- ADD THIS COLUMN ---
  visibility: text('visibility', { enum: ['private', 'public', 'unlisted'] })
    .notNull()
    .default('private'),
});
Here's what these settings mean:

private (Default): The tournament is hidden. Only logged-in members of the organization that created it can see it.

public: The tournament is visible to everyone and will appear in your new "Public Directory."

unlisted: (A "nice-to-have" feature) The tournament is hidden from the directory, but anyone with the direct link can view the schedule and standings.

Create a New Public API Endpoint: You need a new API endpoint that is not protected by authentication.

TypeScript

// server/routes.ts

// This endpoint is OPEN to the public. No auth needed.
app.get('/api/public/tournaments', async (req, res) => {
  // Find all tournaments that are marked 'public'
  const publicTournaments = await db.query.tournaments.findMany({
    where: (tournaments, { eq }) => eq(tournaments.visibility, 'public'),
    with: {
      organization: {
        columns: { name: true } // Also send back the org's name
      }
    }
  });
  res.json(publicTournaments);
});
