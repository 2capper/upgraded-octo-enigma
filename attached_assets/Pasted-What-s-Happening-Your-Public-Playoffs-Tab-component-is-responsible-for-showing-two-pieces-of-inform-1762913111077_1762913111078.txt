What's Happening: Your Public Playoffs Tab component is responsible for showing two pieces of information:

The Playoff Bracket

The Final Pool Standings (that led to the bracket)

The Bug: The component is successfully fetching the playoff games (which is why the bracket renders), but it is not fetching the final, calculated standings.

The Symptom: Because it has no standings data, it's defaulting to a 0-0-0 record for every team.

The Root Cause: Calculating standings is complex. It involves your tie-breaker logic (SP11.2) and must happen on the backend. Your Public Playoffs Tab component is simply forgetting to call the useQuery hook that fetches this calculated standings data.

Here is what the fix will look like:

TypeScript

// In your PublicPlayoffsTab.tsx (or similar component)
import { useQuery } from '@tanstack/react-query';
import { Loader2 } from 'lucide-react';
// Import your components
import { PublicBracket } from './PublicBracket';
import { PublicStandingsTable } from './PublicStandingsTable';

export function PublicPlayoffsTab({ tournamentId }) {

  // You probably already have this, which is for the BRACKET:
  const { data: playoffGames = [], isLoading: isLoadingGames } = useQuery<Game[]>({ 
    queryKey: ['/api/tournaments', tournamentId, 'games'],
    select: (allGames) => allGames.filter(g => g.isPlayoff) 
  });

  // --- THIS IS THE FIX ---
  // You are MISSING the query for the calculated STANDINGS.
  // We must add it.
  const { data: finalStandings = [], isLoading: isLoadingStandings } = useQuery({ 
    // This query key *must* match the one your main "Standings" tab uses
    // to ensure the data is cached and shared.
    queryKey: ['/api/tournaments', tournamentId, 'standings'] 
    // You may have a /standings endpoint, or you may be re-using your
    // complex useStandings() hook here.
  });
  // --- END FIX ---

  if (isLoadingGames || isLoadingStandings) {
    return (
      <div className="flex justify-center items-center py-12">
        <Loader2 className="w-8 h-8 animate-spin" />
      </div>
    );
  }

  // Now you can pass the correct data to the correct components
  return (
    <div className="space-y-8">
      {/* This component now receives the real, calculated standings */}
      <PublicStandingsTable standings={finalStandings} />

      {/* This component gets the playoff games */}
      <PublicBracket bracketGames={playoffGames} />
    </div>
  );
}
